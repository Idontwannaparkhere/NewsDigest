# 财政科学管理资讯搜索

搜索近30天全国财政科学管理改革试点相关资讯，按12项试点任务分类，生成汇总报告。

## 搜索策略

1. **微信公众号**: 37个财政官方公众号 + 财政媒体公众号
2. **省级财政厅官网**: 31个省级财政厅官网新闻动态栏目
3. **关键词匹配**: 基于12项财政科学管理试点任务关键词
4. **时间范围**: 近30天文章
5. **搜索方式**: 
   - 公众号: 公众号名 + 关键词 + 年份 组合搜索（搜狗微信）
   - 官网: 自动探测新闻栏目 + 多模板解析列表页

## 数据来源

### 微信公众号 (✓ 标记)
- 33个省级财政厅公众号
- 20个重点城市财政局公众号
- 通过搜狗微信搜索获取

### 财政媒体公众号 (✬ 标记)
- 财政科学、中国财政、新理财等专业媒体
- 发布财政政策解读和学术文章

### 省级财政厅官网 (◆ 标记)
- 31个省级财政厅官网，配置见 `official_websites.yaml`
- 自动探测"新闻动态"、"工作动态"、"通知公告"等栏目
- 多模板适配解析不同网站结构
- 自动检测网页编码，避免乱码

## 文件说明

| 文件 | 说明 |
|------|------|
| `search.py` | 主搜索脚本，整合微信搜索和官网爬取 |
| `crawl_official_websites.py` | 省级财政厅官网爬取脚本 |
| `generate_report.py` | 报告生成脚本，按试点任务分类 |
| `official_websites.yaml` | 31个省级财政厅官网URL配置 |
| `accounts.yaml` | 微信公众号配置 |
| `pilot_keywords.yaml` | 12项试点任务关键词配置 |

## 使用方法

```bash
# 运行完整搜索（微信+官网）
python3 skills/finance-system-news-search/search.py

# 仅爬取官网
python3 skills/finance-system-news-search/crawl_official_websites.py

# 生成报告（使用已有数据）
python3 skills/finance-system-news-search/generate_report.py
```
